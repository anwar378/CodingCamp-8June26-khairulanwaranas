# Product Overview

**Expense & Budget Visualizer** — a lightweight, client-side web app for tracking personal expenses and visualizing spending against category budgets. No account, no server, no setup required.

## Core Features
- Set a monthly budget per spending category
- Log expenses with description, amount, category, and date
- Visual progress bars showing budget usage per category (safe / warning / over)
- SVG donut chart showing spending breakdown across categories
- Summary dashboard: total budget, total spent, remaining balance, transaction count
- Filter and sort expense history
- All data persisted in browser LocalStorage — works offline
- Confirm modal before destructive actions (delete, clear all)
- Toast notifications for user feedback

## Categories
Food, Transport, Housing, Health, Entertainment, Shopping, Education, Other

## Target User
Anyone who wants a fast, private, no-login budget tracker that runs entirely in the browser.
